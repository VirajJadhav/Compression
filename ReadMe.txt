This folder contains files (C project) to Compress a file using two algorithms :-
	1. Huffman Algorithm.
	2. LZW (Lempel-Ziv-Welch) Algorithm.

To run the project :-
1. Compile just the 'main.c' file.
2. Follow the syntax prompted when you run this file.
3. And for decompression follow the same decompression algorithm command you used for compresssion.
